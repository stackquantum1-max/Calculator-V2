Modern JavaScript Calculator

Description:
This is a fully functional, web-based calculator built with vanilla JavaScript, HTML, and CSS. Designed with a clean and modern user interface, it supports all standard arithmetic operations, including addition, subtraction, multiplication, and division. The calculator features responsive design, real-time input updates, formatted number display, and error handling for edge cases such as division by zero.


Features:

Addition, subtraction, multiplication, and division operations

Real-time number formatting for enhanced readability

Clear (AC) and delete (backspace) functionality

Responsive design compatible with desktop and mobile screens

Modular, clean, and maintainable JavaScript architecture

Easily extensible for additional features or UI enhancements


Purpose:
This project demonstrates a professional approach to DOM manipulation, event handling, and class-based JavaScript design, making it an excellent reference for beginners and intermediate developers looking to build interactive web applications.


Technologies Used:

JavaScript (ES6+) – Class-based structure and modular code

HTML5 – Semantic markup for accessibility

CSS3 – Modern styling, responsive layout, and hover/active effects
